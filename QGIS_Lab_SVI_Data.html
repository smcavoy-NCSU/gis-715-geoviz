SVI Data Using QGIS Effects

This visualization was created using the QGIS blending modes, changing opacity, and creating a shapeburst effect layer. The Social Vulnerability Index data is reported at the census tract level, which is smaller than the US County level. The US Counties are overlaid on top of the Census Tract layer and the blending mode set to multiply. I also added a shapeburst effect to have a faid glow around each of the county boundaries.
