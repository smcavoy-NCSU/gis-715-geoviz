Neon Effect

The Live Layer effects in QGIS were used to create a neon effect for the lines on this map. The Outer Glow effect was used as well as changing the flight path lines color ramp to a gradient.

View Project

Wood Cut Effect

This map was created using bathymetry data. Each layer is a different depth polygon. An inner shadow effect was used to make it look like a wood cut effect and the land layer blending mode was changed to multiply.

View Project

Blending Modes

The Blending Modes options were used in QGIS to make this map. The Land Ownership polygon layer blending mode was changed to Multiply and the PLSS layer blending mode was changed to Overlay.

View Project

Road Network

The road network layer was changed to have an Addition blending mode so that th emost dense areas become white.

USA Shapeburst

This map was made by changing the symbology on a US map. A shapeburst layer was created and a random dot boundary was created.
